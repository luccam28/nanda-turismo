Lei Geral de Proteção de Dados (LGPD)

Nossa conformidade com a Lei nº 13.709/2018

O que é a LGPD?

A Lei Geral de Proteção de Dados (Lei nº 13.709/2018) é a legislação brasileira que regula o tratamento de dados pessoais. Ela estabelece direitos para os titulares de dados e obrigações para as organizações que coletam e processam essas informações.

Nossa Adesão à LGPD

A Nanda Turismo está comprometida com a proteção de dados pessoais e em total conformidade com a LGPD. Implementamos políticas, processos e controles técnicos para garantir que o tratamento de dados seja realizado de forma legal, justa e transparente.

Princípios que Seguimos

Finalidade

Coletamos dados apenas para finalidades específicas e legítimas

Transparência

Comunicamos de forma clara como tratamos seus dados

Segurança

Implementamos medidas técnicas para proteger seus dados

Minimização

Coletamos apenas dados necessários para as finalidades declaradas

Base Legal para o Tratamento

Realizamos o tratamento de dados pessoais com base em:

  • Consentimento: Quando você autoriza expressamente o tratamento
  • Execução de contrato: Para fornecer os serviços de viagem contratados
  • Obrigação legal: Para cumprir determinações legais ou regulatórias
  • Interesse legítimo: Para atividades essenciais ao nosso negócio, respeitando seus direitos

Seus Direitos como Titular

Conforme a LGPD, você tem os seguintes direitos:

Direito Descrição
Confirmação Confirmar a existência de tratamento de seus dados
Acesso Acessar seus dados pessoais tratados por nós
Correção Corrigir dados incompletos, inexatos ou desatualizados
Eliminação Solicitar a eliminação de dados tratados com seu consentimento
Portabilidade Solicitar a transferência de dados a outro fornecedor
Revogação Revogar o consentimento a qualquer momento

Como Exercer Seus Direitos

Para exercer qualquer um dos direitos acima, entre em contato com nosso Encarregado de Proteção de Dados (DPO) através dos canais oficiais. Responderemos sua solicitação no prazo estabelecido pela LGPD.

Medidas de Segurança Implementadas

Para proteger seus dados, implementamos:

  • Criptografia de dados em trânsito e em repouso
  • Controles de acesso baseados em função (RBAC)
  • Monitoramento contínuo de sistemas
  • Políticas de senha seguras
  • Backups regulares e seguros
  • Treinamento regular da equipe em proteção de dados

Transferência Internacional de Dados

Como agência de viagens internacional, podemos transferir dados pessoais para outros países quando necessário para fornecer nossos serviços (ex: reservas com hotéis ou companhias aéreas estrangeiras). Essas transferências são realizadas com garantias adequadas de proteção de dados, conforme exigido pela LGPD.

Incidentes de Segurança

Em caso de incidente de segurança que possa resultar em risco ou dano relevante para você, comunicaremos o ocorrido e tomaremos as medidas necessárias, conforme determina a LGPD e regulamentações aplicáveis.
